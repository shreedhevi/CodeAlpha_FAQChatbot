# CodeAlpha_FAQChatbot🤖

This is a simple FAQ Chatbot built using Python and Flask.  
It responds to user questions based on predefined answers.

## 💡 Features
- Answers frequently asked questions
- Simple and clean user interface
- Fast and easy interaction
- Works based on keyword matching

## 🛠 Technologies Used
- Python
- Flask
- HTML/CSS

## 🚀 How to Run
1. Install required libraries:
   pip install flask

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000

## 📌 Example
User: What is AI?  
Bot: AI means Artificial Intelligence

## 📂 Project Structure
- app.py → Backend logic using Flask  
- templates/ → HTML files for UI
